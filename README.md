# Generador-de-contrase-as-Primer-proyecto-UIDE-
Estoy iniciando en la programación y gracias a las bases adquiridas en la UIDE aquí voy con mi primer proyecto. 
Proyecto Generador de Contraseñas.
Objetivo del Código
El objetivo de este programa es proporcionar una herramienta gráfica que permita a los usuarios generar contraseñas seguras y aleatorias mediante una interfaz de usuario sencilla. Esto ayuda a los usuarios a crear contraseñas que cumplan con los estándares de seguridad, protegiendo así sus cuentas y datos personales.

Características
1.-Genera contraseñas aleatorias con longitud configurable.
2.-Soporta caracteres alfanuméricos y símbolos especiales.
3.-Opción para incluir o eliminar mayúsculas y minúsculas.
4.-Interfaz de línea de comandos sencilla.

Principales Funciones.
1.-Ingresar longitud de la contraseña deseada.
2.-Incluir mayúsculas.
3.-Incluir minúsculas.
4.-Incluir números.
5.-Incluir caracteres especiales.
6.-Generar contraseña.

Modo de uso.
Los usuarios pueden abrir la aplicación, ingresar la longitud deseada, seleccionar los tipos de caracteres que quieren incluir y hacer clic en "Generar contraseña" para obtener una contraseña segura de acuerdo a sus preferencias.
